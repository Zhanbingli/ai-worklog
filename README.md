# AI Worklog

This repository stores concise, public-safe records of AI-assisted work.

It intentionally does not store raw chat transcripts, full prompts, private audit logs, tokens, secrets, or personal data.

## Structure

- `ai-log/`: task-level changelog entries.
- `ai-memory/`: durable decisions, pitfalls, and reusable prompt patterns.
- `.ai-raw/`: reserved for private audit material and ignored by git.
